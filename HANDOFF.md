# Retired Handoff Notes

This file used to contain a Claude-specific, one-time handoff checklist for
the original Hyper-V lab.

It is retired because the project is moving to a multi-repository,
multi-agent workflow:

- `lab-kit`: reusable lab orchestration
- `lab-router`: reusable simple router appliance
- `samba-addc-appliance`: Samba appliance and Samba-specific tests

Use these maintained docs instead:

- `README.md`
- `AGENTS.md`
- `docs/AGENTIC-DEVELOPMENT.md`
- `docs/LAB-TESTING.md`
- `docs/REPO-SPLIT.md`

Historical lab observations remain in `test-results/REPORT.md` and
`test-results/NOTES.md`.
